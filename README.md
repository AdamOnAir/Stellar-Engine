# Stellar Engine

Low-level, Cross-Platform game engine written in portable C (C99).

## Features

- Multiplayer (offline)
- Rendering (2D, 3D)
- Audio
- Lua script

## Supportted platforms

- Windows
- Raspberry Pi
- GNU/Linux

### Backends

- [VDL](https://github.com/AdamOnAir/vdl)
- [miniaudio](https://miniaud.io)
- [minilua](https://github.com/edubart/minilua)
